# Project 5 Presentation Package

Open `project5_presentation_10slides.html` in a browser to present. This is the recommended final deck for a shorter oral defense.

`project5_presentation_showcase.html` is the longer 16-slide deck kept as a detailed backup.

`project5_presentation_creative.html` and `project5_presentation.html` are older versions kept as backup.

Controls:

- Right arrow or space: next slide
- Left arrow: previous slide
- `N`: toggle speaker notes
- Browser print: print/export slides to PDF

Files:

- `project5_presentation_10slides.html`: recommended compact 10-slide defense deck with merged slides and clearer interpretation notes.
- `project5_presentation_10slides_portable.html`: single-file portable HTML with all figures embedded; use this if sending the HTML to another PC.
- `project5_presentation_10slides_portable.pdf`: portable 10-page PDF deck with all figures embedded.
- `project5_presentation_showcase.html`: longer 16-slide backup deck with more figures and short interpretation paragraphs under each image.
- `project5_presentation_creative.html`: previous creative backup deck.
- `project5_presentation.html`: simpler backup deck.
- `figure_interpretation_guide.md`: short guide explaining what each figure means, including why SHAP and SmoothIG raw values should not be compared directly.
- `oral_script_10slides.md`: oral script aligned with the compact 10-slide deck.
- `script_oral_10slides_fr.md`: French oral script aligned with the compact 10-slide deck.
- `script_oral_10slides_fr_detaille.md`: detailed French oral script with stronger concept explanations, especially for slides 5 to 10.
- `script_oral_10slides_fr_detaille_v2.md`: clearer adapted extended oral script using French explanations with standard ML/security terms kept in English.
- `analyse_figures_fr.md`: French figure interpretation guide, including the SHAP vs SmoothIG scale explanation.
- `questions_reponses_defense_fr.md`: French defense Q&A adapted to the project and course concepts.
- `questions_reponses_cours_projet_fr_v2.md`: improved French Q&A connecting the course concepts to the project with clearer terminology and stronger defense answers.
- `document_preparation_personnel_fr.md`: private French study guide explaining the project concepts, choices, metrics, attacks, defenses, results, and limitations in depth.
- `document_preparation_personnel_fr_v2.md`: clearer French/English-terms study guide with detailed preprocessing, model-by-model explanations, XAI/stability examples, Security-OR explanation, and `pp` definition.
- `oral_defense_script.md`: full slide-by-slide oral script.
- `oral_defense_qa.md`: likely oral-defense questions and answers.
- `two_author_oral_script_and_qa_detailed.md`: longer two-author oral defense script with detailed result analysis, deductions, limitations, and extended Q&A.
- `course_adapted_oral_defense_qa.md`: Q&A adapted to the course PDFs in `cours/`, using class concepts such as MLPs, activation/loss functions, gradient descent, CNNs, RNNs, autoencoders, GANs, and transformers.

The decks use figures from `../../outputs_cuda/`, so keep this folder inside `new/presentation/` unless you update the image paths.
